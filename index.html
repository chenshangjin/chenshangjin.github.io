如何用 Hexo + Github Page 搭建个人博客

Hexo 是一个快速、简洁且高效的博客框架。

GitHub Pages是GitHub提供的一项功能，允许用户在GitHub上托管和发布静态网站。它的好处是不需要处理服务器和域名等繁琐的事情就能展示自己的个人博客。

现在您看到的这篇文章就是基于Hexo框架在Github Pages上搭建的个人博客。下面把搭建过程记录下来，总体来说还是比较简单的。但是要记住，个人博客最重要的还是内容，形式是次要的！

一.Hexo环境搭建

1.安装node.js

安装过程略。安装完成后，运行node --version检查node.js版本号，如果正常显示版本号说明安装成功。

2.安装Git

安装过程略。安装完成后，运行git --version检查Git版本号，如果正常显示版本号说明安装成功。

3.设置npm镜像源

npm config set registry http://mirrors.cloud.tencent.com/npm/

4.使用npm安装Hexo

npm install -g hexo-cli

二.GitHub新建仓库

在Github上新建public仓库chenshangjin.github.io，注意这里的仓库名的第一部分必须是精确匹配Github用户名。

三.部署Hexo

1.新建hexo项目

cd blog

hexo init

npm install

2.新建文章

hexo new "如何用 Hexo + Github Page 搭建个人博客"

编辑完成后，在本地预览

hexo s
